How It Works

This implementation uses a semi-transparent black overlay to darken the background image while keeping the content completely clear:

/* HTML Structure */ <div class="background-container"> <div class="background-image"></div> /* Background image */ <div class="dark-overlay"></div> /* Dark overlay */ <div class="content"> /* Content container */ <div class="text-content"></div> /* Text content */ <div class="search-box"></div> /* Search box */ </div> </div> /* CSS for darkening effect */ .dark-overlay { background: rgba(0, 0, 0, 0.7); /* Black with 70% opacity */ } /* Content positioned above */ .content { position: relative; z-index: 3; /* Higher than background layers */ } .text-content, .search-box { /* No background property - remains clear */ }

To use your own image, replace the background URL in the CSS with your image path:

.background-image { background: url('localbus.jpeg') center/cover; }

Adjust the darkness by changing the opacity value (0.7 in the example):

.dark-overlay { background: rgba(0, 0, 0, 0.5); /* Lighter - 50% opacity */ } .dark-overlay { background: rgba(0, 0, 0, 0.8); /* Darker - 80% opacity */ }
