How to Implement This Solution

To shade the entire background container while keeping content clear, use the ::before pseudo-element:

/* Main container with background image */ .bus-tracking { padding: 80px 0; background: url('localbus.jpeg') center/cover; color: white; position: relative; /* Required for pseudo-element positioning */ } /* Dark overlay for the entire container using ::before */ .bus-tracking::before { content: ''; position: absolute; top: 0; left: 0; width: 100%; height: 100%; background: rgba(0, 0, 0, 0.6); /* Adjust opacity as needed */ z-index: 1; /* Places overlay above background but below content */ } /* Content container - positioned above the overlay */ .bus-tracking-content { position: relative; z-index: 2; /* Higher than the overlay */ }

To use your own image, replace the background URL:

.bus-tracking { background: url('localbus.jpeg') center/cover; }

Adjust the darkness by changing the opacity value:

.bus-tracking::before { background: rgba(0, 0, 0, 0.4); /* Lighter overlay */ } .bus-tracking::before { background: rgba(0, 0, 0, 0.7); /* Darker overlay */ }
